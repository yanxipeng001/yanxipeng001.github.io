##使用Go编写第一行代码

在新建golangFile文件夹，在文件夹中新建main.go，在里面写入如下代码

编译

go build main.go

或者直接编译当前包含go代码的文件夹

go build

会看到编译后文件夹内多出

main.exe或者golangFile.exe

直接在终端执行就会看到“hello world”了

.\main.exe

如果想查看编译过程

go build -x main.go

指定一个编译后的二进制文件名称

go build -x -o hello.exe main.go

如果省去手动编译过程，一步编译+执行

go run main.go 这样直接在一个其它临时的目录生成.exe并执行出结果
